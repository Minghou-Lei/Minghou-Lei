## About Me

I am a game engine developer focused on **real-time rendering**, **engine systems**, and **graphics tooling**.

My daily work mainly involves **Unity**, **Unreal Engine**, **C#**, **C++**, **HLSL**, **ShaderLab**, and **Python**.  
I care about building rendering and engine workflows that are practical, debuggable, scalable, and production-oriented.

---

## Focus Areas

- **Real-time Rendering**  
  Shadow pipelines, depth / occlusion workflows, GPU-driven rendering, clustered lighting, shader systems.

- **Game Engine Development**  
  Unity URP / HDRP, Unreal Engine 5, custom engine infrastructure, rendering feature integration.

- **Graphics Tooling**  
  Shader debugging, RenderDoc / PIX workflows, asset import pipelines, DDS / texture migration, automation tools.

- **Engineering Workflow**  
  Windows-first development, JetBrains Rider, build automation, repository-scale code navigation, agent-assisted coding workflows.
